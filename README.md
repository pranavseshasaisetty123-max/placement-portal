# Placement Portal – Recruitment Management System

## Overview

Placement Portal is a full-stack web application designed to streamline the campus recruitment process for students, recruiters, and administrators. The platform enables student profile management, job postings, application tracking, and recruitment workflow management through a centralized dashboard.

## Features

- Student Registration and Authentication
- Recruiter and Administrator Login
- Job Posting Management
- Student Profile Management
- Application Tracking System
- Database-Driven Recruitment Workflows
- Secure Session Handling
- Responsive Web Interface
